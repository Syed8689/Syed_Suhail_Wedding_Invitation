## GitHub Pages Deployment

### Step 1

Create a new GitHub repository.

Example:

wedding-invitation

### Step 2

Upload:

- index.html
- styles.css
- script.js
- config.json
- assets folder

### Step 3

Open:

Settings → Pages

### Step 4

Configure:

Source → Deploy from branch

Branch → main

Folder → /root

Save

### Step 5

GitHub generates:

https://YOUR_USERNAME.github.io/wedding-invitation/
